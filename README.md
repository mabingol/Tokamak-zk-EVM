# Tokamak-zk-EVM

- `packages/buildQAP`: QAP Build Tool (JS + Shell + CIRCOM)
- `packages/mpc`: MPC Implementation (Rust + Solidity)
- `packages/tokamak-zk-snark`: Tokamak zk-SNARK Implementation (Rust + Solidity)

## Getting Started
